# Visual Elements Reference

## Platform Designs

### Standard Isometric Platform
- Rectangular base with beveled edges
- Soft blue/cyan edge glow
- Gradient surface: light gray to white
- Height: subtle 3D depth effect

### Elevated Platform (for emphasis)
- Raised platform with visible support structure
- Multiple tier levels possible
- Stronger glow effects for importance

### Circular Platform
- Use for central hub elements
- Rotating ring accents optional
- Holographic projection effect above

## Connection Styles

### Data Streams
- Flowing particles along path
- Gradient glow (cyan to purple)
- Animated appearance (for video)
- Varying thickness for hierarchy

### Circuit Pathways
- PCB-style traces
- Right-angle routing
- Node connection points
- Subtle pulse glow

### Energy Beams
- Straight line connections
- Intense center glow
- Soft edge diffusion
- Color: cyan or electric blue

### Organic Flows
- Curved, flowing paths
- Multiple parallel lines
- Gradient opacity
- Ethereal quality

## Background Elements

### Circuit Board Pattern
- Subtle traces in darker blue
- Occasional bright nodes
- Grid alignment
- Low opacity (10-20%)

### Hexagon Grid
- Tech/futuristic feel
- Varying opacity cells
- Some cells highlighted
- Depth through size variation

### Particle Field
- Scattered small dots
- Random distribution
- Varying brightness
- Creates depth illusion

### Gradient Vignette
- Darker corners
- Focus on center
- Subtle radial gradient
- Professional finish
